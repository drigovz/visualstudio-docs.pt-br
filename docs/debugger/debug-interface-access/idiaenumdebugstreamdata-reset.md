Redefine para o início de uma sequência enumerado.  
  
## <a name="syntax"></a>Sintaxe  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Retorna `S_OK`.
