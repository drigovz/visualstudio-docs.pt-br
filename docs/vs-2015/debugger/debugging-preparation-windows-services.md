# <a name="debugging-preparation-windows-services"></a>Preparação de depuração: Serviços Windows
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Um serviço do Windows é um programa executado em segundo plano no Microsoft Windows. Os exemplos incluem o serviço de Telnet e o serviço de tempo do Windows, que atualiza o relógio visível do computador. Um serviço do Windows não pode ser executados no [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]; ele deve ser executado no contexto do Gerenciador de Controle de Serviços. Para obter mais informações, consulte [criando serviços do Windows](http://msdn.microsoft.com/library/0f5e2cbb-d95d-477c-b2b5-4b990e6b86ff), [depuração de aplicativos de serviço do Windows](http://msdn.microsoft.com/library/63ab0800-0f05-4f1e-88e6-94c73fd920a2), e [aplicativos de serviço do Windows](http://msdn.microsoft.com/library/ba72d648-9553-4849-b829-069ad5ea014b).
