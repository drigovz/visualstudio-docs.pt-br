# <a name="getting-started-with-source-control-vspackages"></a>Introdução ao VSPackages de controle do código-fonte
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Um VSPackage de controle do código-fonte é uma alternativa mais avançada para o plug-in de controle do código-fonte. (Consulte [criando um plug-in de controle do código-fonte](../../extensibility/internals/creating-a-source-control-plug-in.md) para obter mais informações sobre a fonte de plug-ins de controle.) Controle completo sobre o modelo de controle do código-fonte, recursos e interface do usuário (UI) permite a um VSPackage de controle de origem, e ele se integra a [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] ambiente como um VSPackage.
