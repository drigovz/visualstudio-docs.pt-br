# <a name="asynctaskmethodbuilderlttresultgtmtask-field"></a>AsyncTaskMethodBuilder&lt;TResult&gt;. m_task campo
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Representa a inicialização ociosa criado a tarefa.  
  
 **Namespace:** <xref:System.Runtime.CompilerServices?displayProperty=fullName>  
  
 **Assembly:** mscorlib (em mscorlib. dll)  
  
 Porque você não pode acessar esse membro interno do .NET Framework, a sintaxe a seguir é fornecida em comum Intermediate Language (CIL).

```  
.field private class System.Threading.Tasks.Task`1<!TResult> m_task  
```
