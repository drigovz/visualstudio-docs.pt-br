# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28304: o parâmetro de referência de C++ < parameter_name >, um posicionado `_Notref_` operador foi encontrado aplicado ao \<token >.  
  
 O `_Notref_` operador deve ser usado somente em circunstâncias especiais que envolvem parâmetros de referência do C++ e apenas em macros fornecidas pelo sistema. Ele deve ser seguido imediatamente por um `_Deref_` operador ou uma anotação de funcional.
