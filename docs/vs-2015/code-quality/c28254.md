# <a name="c28254"></a>C28254
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28254: não há suporte para dynamic_cast <> () em anotações  
  
 O C++ `dynamic_cast` operador não pode ser usado em anotações.
